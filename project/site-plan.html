Site Plan – LinkCar Shuttles

“Luxury Transport & Exclusive Tours in Cape Town”

1. Site Name

LinkCar Shuttles
This name highlights the link between premium shuttle transport and exclusive Cape Town experiences.

2. Site Purpose

The website promotes premium shuttle services and luxury tours in Cape Town, allowing customers to learn, explore, and book online.

3. Target Audience

  • International Tourists
  • Corporate Travelers
  • High-net-worth Locals

4. Content Outline

  1. Home Page
  2. Services Page
  3. Tour Packages Page
  4. Booking Page
  5. About Us Page
  6. Contact Page

5. Scenarios

  • Business traveler needing airport transfer.
  • Tourist couple booking a wine tour.
  • Executive needing chauffeur service.

6. Color Schema

Colors: Black, Gold (#D6A463), Navy (#253A47), White.

7. Typography

Headings: Elegant Serif
Body: Modern Sans-Serif

8. Wireframes

Mobile: Hamburger menu, hero image, scrollable cards.
Desktop: Horizontal nav, full-width hero, grid layout services.
